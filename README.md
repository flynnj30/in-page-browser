# In-Page Browser with Node.js Proxy

A full-featured in-page browser that uses a Node.js proxy server to bypass iframe restrictions and display any website.

## Features

- ✅ **Proxy Server**: Fetches and modifies websites to work inside an iframe
- ✅ **Full Browsing**: Navigate, go back/forward, reload
- ✅ **Search**: Google search integration
- ✅ **Bookmarks**: Save and manage favorite websites
- ✅ **URL Bar**: Smart URL parsing and formatting
- ✅ **Secure/Insecure Indicators**: Visual security status
- ✅ **Keyboard Shortcuts**: Ctrl+L, Ctrl+R, Alt+Left/Right
- ✅ **Responsive Design**: Works on desktop, tablet, and mobile
- ✅ **Error Handling**: Graceful error pages and recovery
